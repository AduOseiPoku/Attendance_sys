I'm giving you the user flow of the app. The user scan the QR code, they get a page that ask if they are a member or they are yet to visit. When they select a member and enter their details, it is check from the database to see if the details exist. If it doesn't exist then it will ask them to join. As the user is typing their name, there will be a filter that will check their name as they enter, so when they select a name that is in the database, it will automatically fill the rest of the form for them.
